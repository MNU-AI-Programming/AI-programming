## 📌 Course Overview

본 교과목은 **Python 프로그래밍 기초 → 데이터 분석 → 시각화 → 동시성
프로그래밍**까지 단계적으로 학습합니다.\
또한 대규모 언어모델(LLM)을 학습 도구로 활용하여 **With AI 프로그래밍
역량**을 기릅니다.

> AI는 도구이며, 최종 검증은 사람이 수행합니다.

------------------------------------------------------------------------

## 🎯 Learning Objectives

수강 후 다음을 수행할 수 있어야 합니다:

-   Python 기본 문법으로 프로그램 작성
-   파일 입출력 및 CSV/JSON 처리
-   NumPy / Pandas 데이터 분석
-   Matplotlib 시각화 및 해석
-   threading 기반 동시성 프로그램 구현
-   LLM을 활용한 코드 생성·디버깅·리팩터링 및 검증
